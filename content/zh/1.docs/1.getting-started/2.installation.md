## 普通用户

BlueScript 以网络应用程序的形式提供，您只需使用**浏览器**，即可随时随地轻松使用，无需任何安装过程。 访问官方网络应用程序，立即开始使用 BlueScript 的所有功能。

## 自助托管

除官方网络应用程序外，您还可以在自己的服务器上托管**BlueScript**。 该选项适合具有服务器管理技能或喜欢定制环境的用户。

### 使用 Docker 托管

https://hub.docker.com/r/suhjae/bluescriptBlueScript 可使用 [官方 Docker 镜像]（ ）轻松托管。 请按照以下步骤安装 Docker 并运行 BlueScript 服务器。

#### **1. 安装 Docker**

https://www.docker.com/如果尚未安装 Docker，请从[Docker 官方网站]（ ）下载并安装适用于你的平台的 Docker。

#### **2. 调出 BlueScript Docker 映像**

安装好 Docker 后，在终端运行以下命令下载 BlueScript 官方镜像：

```bash
docker pull suhjae/bluescript
```

#### **3. 运行 BlueScript**

下载 Docker 映像后，使用以下命令启动容器：

```bash
docker run -d -p 80:3000 --name bluescript suhjae/bluescript
```

给你

- d\` 在后台运行容器。
- 将本地计算机上的端口 80 映射到容器内的端口 3000。

#### \*\*4. 访问 BlueScript

打开浏览器，导航至 `http://localhost` 访问 BlueScript。

#### **5. 配置设置**

要进行其他配置，可以修改 `docker-compose.yml` 文件或使用 Docker 环境变量来定制 BlueScript。 https://traefik.io/traefik/例如，你可以在 `docker-compose.yml` 文件中设置一个类似 [Traefik]() 的网络代理来连接你的域名，并创建一个更高级的设置。
